Java Appointment Scheduler

A Java application for scheduling appointments for existing and new customers using a database.

Technologies used: Java, JavaFX, MySQL, JDBC, SceneBuilder, NetBeans, IntelliJ IDEA.

  • Used the Java Database Connectivity (JDBC) API to manipulate an external SQL Database.
  • ADD, SELECT, MODIFY, and DELETE items from a MySQL database that takes into consideration Foreign Key constraints in the database.
  • Robust Time Zone support. Features automatic timezone conversion and notification generation based on the local machines’ time zone and a calendar that filters appointments by week, month, and year.
  • Includes security through a user login form, using the database to manage valid user accounts.
  • Includes logic checks throughout the entire application to ensure proper entry of data.
  • Automatic filling of TextField and ComboBox fields when MODIFYING data.
  • General User Interface support using JavaFX.
  • Lambda Expressions and Github and Git for Version Control.

Python Predictive Analytic Model

A machine learning predictive analytic model that predicts the price of a vehicle based on historical data from other vehicles.

Technologies used: Python, pyCharm, Jupyter Notebook, Binder.

  • Includes data collection, cleaning, and visualization as part of the machine learning process.
  • Python libraries used include NumPy, Pandas, Matplotlib, Seaborn, Scikit-learn, IPyWidgets, and Plotly.
    • Numpy was used to support Pandas in creating Arrays and Matrices.
    • Pandas was used to create DataFrames and load in our CSV data.
    • Matplotlib was used to visualize the data through histograms, heatmaps, bargraphs, boxplots, etc.
    • Scikit-learn was used to create our supervised machine learning model. In the case of this project, that learning model used Linear Regression.
  • Includes many metrics such as the...
    • Mean Absolute Error (MAE)
    • Mean Squared Error (MSE)
    • Root Mean Squared Error (RMSE)
  • Created in PyCharm and Jupyter Notebooks and deployed onto Binder.
